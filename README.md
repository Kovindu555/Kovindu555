### Featured Projects

- **[Employee Attrition Multiclass Prediction](https://github.com/Kovindu555/Employee-Attrition-Multiclass-Prediction)** — This repository presents a complete Employee Attrition Prediction and Analysis System built for the global healthcare industry. It leverages machine learning, data analytics, and predictive modeling to identify employees who may be at risk of leaving, enabling data-driven workforce planning and proactive employee retention strategies.

- **[Customer Churn Prediction](https://github.com/Kovindu555/Customer-Churn-Prediction)** — This repository contains a comprehensive customer churn prediction project using a telecommunications dataset. It explores a binary classification problem by comparing the performance of a Decision Tree model with a Neural Network model to evaluate their effectiveness in predicting customer churn and supporting customer retention initiatives.

- **[RAG Conversational AI for LMS](https://github.com/Kovindu555/RAG-Conversational-AI-System)** — Developed a Retrieval-Augmented Generation (RAG) conversational AI for a Learning Management System (LMS) that delivers syllabus-specific assistance. The system answers academic queries, generates A-Level multiple-choice and structured questions, and ensures responses remain strictly within the defined educational content.
